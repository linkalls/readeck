# Profil użytkownika

[Sekcja profilu](readeck-instance://profile) Readeck pozwala ci zmienić twoje dane osobowe, hasło i inne ustawienia.

## Edytuj swój profil

Na stronie głównej profilu możesz zmienić nazwę użytkownika, adres email i wybrać język aplikacji.

## Zmień hasło

Na stronie [Hasła](readeck-instance://profile/password) możesz zmienić hasło używane do połączenia się z Readeck.

## Tokeny API

Token API pozwala uzyskać dostęp i używać [API Readeck](readeck-instance://docs/api) do zbudowania czego potrzebujesz. Możesz tworzyć i zarządzać tokenami w sekcji [Tokeny API](readeck-instance://profile/tokens) profilu użytkownika.

Możesz ograniczyć do czego dany token ma dostęp poprzez API i jak długo jest ważny.

## Hasła aplikacji

Jeśli potrzebujesz udzielić dostępu do swojego konta Readeck jakiejś aplikacji lub usłudze, nie możesz udostępnić swojej głównej nazwy użytkownika i hasła; to nie zadziała.

Co możesz zrobić to stworzyć [Hasło aplikacji](readeck-instance://profile/credentials).

Możesz ograniczyć do czego dane hasło aplikacji ma dostęp poprzez API.

Jak już stworzysz hasło aplikacji, możesz go użyć aby uzyskać dostęp do [API Readeck](readeck-instance://docs/api) lub usług eksportu.

Odwiedź stronę pomocy [Katalog E-booków](./opds.md) aby zobaczyć przykład z życia wzięty.

**Uwaga**: nawet jeśli możesz użyć hasła aplikacji to dostępu do API, rekomendowane jest użycie tokena API gdy tylko możliwe.
